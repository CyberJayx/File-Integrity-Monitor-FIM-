# File Integrity Monitor (FIM)

## Overview
[span_4](start_span)[span_5](start_span)This tool is designed for **Digital Forensics** and **Incident Response** workflows[span_4](end_span)[span_5](end_span). it monitors critical system files for unauthorized changes by comparing SHA-256 cryptographic hashes.

## Features
- **Hashing:** Uses the `hashlib` library to generate unique fingerprints for files.
- **Change Detection:** Alerts the user if a file has been modified, a key step in detecting malware or unauthorized access.

## Technical Skills
- Cryptography, Forensic Triage, Incident Response.
